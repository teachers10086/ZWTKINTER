## 🎯 核心特性

- 🀄 **全中文API** - 函数/参数全中文命名
- 🧩 **模块化设计** - 即插即用的组件体系
- 🎨 **主题支持** - 内置多套现代UI主题
- 📱 **响应式布局** - 自动适配不同屏幕尺寸
- 🔌 **扩展生态** - 支持常用插件和扩展

## 🚀 5分钟入门

```python
from ZWTK import *

# 创建主窗口
窗口 = 主窗口(标题="我的应用", 宽=800, 高=600)

# 添加组件
标签(窗口, "欢迎使用ZWTK!", 行=0, 列=0)
输入框 = 输入框(窗口, 行=1, 列=0)

def 提交():
    弹窗("输入内容", 输入框.获取文本())

按钮(窗口, "提交", 提交, 行=2, 列=0)

# 运行程序
运行(窗口)
```

## 🪟 窗口管理

### 创建窗口
```python
窗口 = 主窗口(
    标题="演示程序",
    宽=1024,
    高=768,
    背景色="#f0f0f0"
)
```

### 窗口操作
| 方法                | 说明                 |
|---------------------|----------------------|
| `窗口.置顶()`       | 设置窗口始终在最前   |
| `窗口.全屏()`       | 切换全屏模式         |
| `窗口.居中()`       | 窗口居中显示         |
